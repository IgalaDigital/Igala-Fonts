Installation

Overview

Thank you for joining the awesome people using the IgalaFont to build and develop awesome Igala Contents.
Just like every other font, the IgalaFont can be use everywhere and for different purposes ranging from presentation, writing & designing etc. It's left for you to start writing awesome Igala Contents.

Getting the fonts to work on your machine won't take 3mins if the installation procedure below are been followd correctly.

Automatic Installation

If you're already familiar with installing fonts, this will come very easy just as before.
This will get the font working on your system without you making any settings.

Steps
  • Download the font using the Download Fonts button on our web
  • Unzip/Extract the Downloaded file Igala-Font.zip into any folder
  • If extraction done, Open the folder and double click the files which you should see something like the below
  • A new window will pop out, click install at the right hand side just as you see from the image above
  • You're done. Start Writing today.
    Note: Check the how to use section to learn how to write with it if it's your first time.

Manual Installation

Unlike the the automatic way of installation, this way is a bit technical any may cost you roughly 5mins of your time depending on how boyaunt you are using computers. At the same time. you may consider this as the automatic one if you know your way around.

Steps
  • Download the font using the Download Fonts button on our web
  • Unzip the Downloaded file Igala-Font.zip into any folder
  • If extraction done, Open the folder and select all the files
  • Copy them all
